- `pending` - Not yet processed
- `in_progress` - Currently researching
- `done` - SQL generated, added to import set

## Import Column
- `yes` - Capoeirista or direct lineage figure, should be imported
- `?` - Needs investigation to determine if they qualify

**Note:** Persons with `Import = no` (non-capoeiristas like historians, military figures, writers) should NOT appear in this backlog - they don't belong in the genealogy data model.
