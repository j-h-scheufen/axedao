# Spiritual/Religious Influences Backlog

This backlog tracks spiritual and religious figures, terreiros, and sacred spaces that have documented connections to capoeira mestres and groups. These entities are tracked for potential future data model expansion.

## Status Legend
- `pending` - Not yet processed
- `reviewed` - Evaluated, decision made on import approach
- `imported` - Added to genealogy database

## Import Considerations

These entities represent the spiritual dimension of capoeira lineage:
- **Candomblé**: Ialorixás, Babalorixás, Ogãs, Ekédis, Terreiros
- **Umbanda**: Spiritual leaders, mediums, centros
- **Syncretic**: Catholic saints with capoeira devotions, mixed practices

Future data model may include:
- `genealogy.spiritual_profiles` table
- `spiritual_influence` or `blessed_by` predicates
- Terreiro-to-group and terreiro-to-person relationships
